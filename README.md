# Reviews
Analyse de sentiment et confrontation de classifieurs de Machine Learning sur des critiques vidéoludiques de joueurs (via Steam) et de la presse (via Metacritic).

-----

Objectif : Entraîner des classifieurs afin de déterminer lequel est le plus efficace à la tâche suivante : Détecter les "reviews" positives et négatives dans le corpus des avis Steam (joueurs).
Puis les confronter à l'avis presse (Metacritic) afin de vérifier si ce dernier est coordonné à l'avis des joueurs ou y est opposé.

-----

Contenu : 

* 76 jeux différents
* 1275 reviews
* 7 bases de données
* 4 graphiques d'analyses (Statistiques et Sentiment)
* 21 graphiques d'analyses (Matrices de confusions, graphiques de comparaisons des classifieurs, f-mesure)
* Rapport d'analyse statistique (.txt /.csv)
* 4 rapports textuels résumant les résultats par classifieur
* 8 scripts
